# sijil

Logger using pino

## requirements

* Node.js v22.x, v24.x is recommended
* npm v11.x

## development

```sh
# install Node modules
npm i

# modify code inside ./src

# build
npm run build

# review/modify version number in package.json
# publish
npm publish
```

## usage

```sh
npm i rataqa/sijil
```
